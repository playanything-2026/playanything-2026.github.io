Abstract

Vision-language model (VLM) driven game agents have shown strong performance in specific games; however, existing frameworks require extensive manual customization for each new game, including game-specific prompts and action functions. This dependence results in near-zero cross-game generalization. To address this limitation, we propose PlayAnything, a plug-and-play multimodal prompt adaptation framework that enables VLM-based agents to generalize to diverse unseen games.

Specifically, PlayAnything consists of three complementary modules: i) Gather, a multimodal retriever that constructs structured game profiles from gameplay videos, screenshots, and strategy texts; ii) Adapter, a core module fine-tuned on Qwen3-VL-8B that transforms these profiles into executable pre-filled prompts and action functions; and iii) Guider, a suggestion module that provides targeted improvement suggestions for humans to refine the pre-filled prompts via cross-reference coverage analysis.

Experiments demonstrate that PlayAnything, trained on only 5 games, achieves 100% runnability and an average success rate of 58% across 40 unseen games. It substantially outperforms existing baselines: Cradle and CombatVLA both achieve 0%, while the much larger NitroGen reaches only 42%. These results demonstrate superior generalization with orders of magnitude less training data. Furthermore, with guidance from the Guider module, humans need to make only 136 character-level modifications on average to the pre-filled prompts, which further increases the average success rate to 67.5%. We will open-source all resources.

Framework Overview

Overview of our PlayAnything. 1) Gather retrieves multimodal game information. 2) Adapter produces pre-filled prompts and action functions. 3) Guider produces targeted improvement suggestions for human-AI collaborative refinement.

🔍

Gather

Retrieves gameplay videos, screenshots, and strategy texts from external sources and constructs structured game profiles via a fully automated pipeline.

⚙

Adapter

Fine-tuned on Qwen3-VL-8B-Instruct with only 5 training games. Takes multimodal game profiles as input and generates pre-filled prompts and executable action functions for the target agent framework.

🧭

Guider

Conducts cross-reference coverage analysis between Adapter outputs and raw game profiles. Provides structured improvement suggestions for efficient human-AI collaborative prompt refinement.

  1. PlayAnything overcomes the zero-generalization barrier. Both Cradle and CombatVLA achieve 0% Runnability (i.e., SR1), confirming that modular frameworks and specialized VLAs fail to launch unseen games or transfer skills across types. In contrast PlayAnything achieves 100% Runnability across all 40 test games by automatically generating valid action spaces and prompt structures.
  2. PlayAnything outperforms large-scale foundation models with a fraction of the data. Compared to NitroGen (pre-trained on 40,000 hours of gameplay), our PlayAnything framework (trained on only 5 games) elevates the overall average success rate from 42.0% to 67.5%. This advantage is most pronounced in Coordinate Control games, where NitroGen collapses to near 25.8% due to its rigid gamepad primitives, whereas PlayAnything adaptively grounds interaction logic in visual coordinates to achieve robust generalization.
  3. Guider-directed refinement unlocks peak performance with minimal human effort. With lightweight strategic intervention (median Edit Distance of only 136 characters), PlayAnything further pushes the Level 2 success rate to 92.0%. These results confirm that our framework effectively shifts the human role from redundant programming to high-level strategic correction, allowing the agent to resolve complex semantic bottlenecks that are typically inaccessible to pure end-to-end models.

Conclusion

In this paper, we present PlayAnything, a plug-and-play multimodal prompt adaptation framework designed to enable VLM-driven game agents to generalize across diverse unseen games without requiring labor-intensive per-game manual customization. Experiments show that training on only 5 games, PlayAnything reaches 100% runnability and 58% average success on 40 unseen games, clearly outperforming Cradle, CombatVLA, and the much larger NitroGen. This demonstrates strong cross-game generalization with far less training data. With Guider, only 136 character-level human edits on average further raise success to 67.5%, showing the practicality and efficiency of human-in-the-loop refinement. In future work, we plan to investigate fully autonomous iterative self-optimization and extend the framework to handle more complex 3D gaming environments.
